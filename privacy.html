Privacy Policy for DailyVibe

Last updated: March 2026

1. Overview

DailyVibe is a Chrome extension that provides motivational quotes, focus tracking, and beautiful background images on the new tab page.

2. No Personal Data Collection

DailyVibe does not collect, store, or share any personally identifiable information. We do not track users, browsing activity, or search queries.

3. Local Storage Usage

Any user data such as focus tracking or preferences is stored locally using Chrome's storage API. This data remains on the user's device and is never transmitted to any external server.

4. Third-Party Services

This extension uses the following third-party APIs:

  • Unsplash API – to display background images
  • type.fit API – to display motivational quotes

No personal data is shared with these services. Requests are made only to fetch public content.

Requests to third-party APIs may include basic technical information such as browser type or IP address as part of standard web requests. No personal or identifiable user data is intentionally collected, stored, or shared by the extension.

5. Permissions

The extension uses only the permissions necessary for functionality:

  • storage – to save user preferences locally

6. Security

We take reasonable measures to ensure the extension is secure and does not expose users to unnecessary risks.

7. Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated date.

9. Search Functionality

DailyVibe does not modify, intercept, or track any user search queries. Any search functionality (if present) redirects users directly to third-party search engines without modification.
